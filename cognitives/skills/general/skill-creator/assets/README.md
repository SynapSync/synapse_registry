# Skill Creator Assets

This directory contains templates for the `skill-creator` skill.

## Directory Structure

```
assets/
├── README.md                    # This file
├── SKILL-TEMPLATE-BASIC.md      # Minimal skill template (~100 LOC)
└── SKILL-TEMPLATE-ADVANCED.md   # Full-featured skill template (~250 LOC)
```

## Templates

### SKILL-TEMPLATE-BASIC.md

Minimal template for simple skills. Includes frontmatter, core sections, and basic workflow. Best for skills that don't need modes, helpers, or complex workflows.

### SKILL-TEMPLATE-ADVANCED.md

Full-featured template with assets pattern support, multiple modes, helpers, validators, and comprehensive documentation. Best for complex skills that benefit from modular organization.
